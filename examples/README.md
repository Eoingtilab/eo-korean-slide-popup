# Examples

This directory contains reusable popup content examples for EO Korean Slide Popup.

## Website open popup

Use `website-open-popup.html` as a simple HTML slide for announcing a website renewal or grand opening.

Recommended popup settings:

- Slide type: HTML
- Position: Center
- Width: 640 to 720px
- Overlay: On
- Title bar: Optional
- Hide today button: On

The sample uses the `.eoksp-open-popup` CSS class. You may paste the sample into the HTML slide editor and edit the text from the WordPress admin screen.
