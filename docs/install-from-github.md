# Installing from GitHub

GitHub's default download ZIP may contain a root folder named `eo-korean-slide-popup-main`.

For WordPress plugin upload, the ZIP should contain this root folder instead:

```text
eo-korean-slide-popup/
```

If WordPress shows two copies of the plugin, deactivate both, delete the duplicate `eo-korean-slide-popup-main` copy, and upload a correctly packaged ZIP.
